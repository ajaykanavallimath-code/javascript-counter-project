# 🧮 Counter App

A simple and interactive **Counter Application** built using **HTML, CSS, and JavaScript**.  
This project demonstrates core JavaScript concepts like **DOM manipulation**, **event handling**, and dynamic UI updates.

---

## 🚀 Features

- ➕ Increase counter value  
- ➖ Decrease counter value  
- 🔄 Reset counter to zero  
- 🎨 Modern gradient background  
- 💻 Clean and responsive UI  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Gradient + Flexbox)  
- JavaScript
